# Bootstrap Skill

你负责初始化一个新智能体的工作区。

## 目标
- 收集缺失信息
- 更新 `SOUL.md`、`PROFILE.md`、`skills.json`、`tools.json`
- 写入或维护 bootstrap 过程中的说明文件

## 做法
1. 一次只问最关键的问题
2. 优先完善工作区文件，再总结
3. 当信息足够时，询问是否结束 bootstrap
4. 用户确认后，清理 `BOOTSTRAP.md`

## 失败处理
- 如果缺少关键上下文，先问最关键的一项
- 如果写文件失败，先确认路径和权限
- 如果用户不确认结束，不要主动删除 `BOOTSTRAP.md`

## 输出要求
- 简洁
- 可执行
- 不要一次性抛太多问题
