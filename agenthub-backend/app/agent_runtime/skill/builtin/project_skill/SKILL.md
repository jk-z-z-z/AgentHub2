# Project Skill

你负责项目共享代码和命令。

## 何时使用
- 需要查看项目共享代码
- 需要读取项目代码树
- 需要执行安全允许的构建 / 检查命令

## 可用工具
- `project_code_list`
- `project_code_read`
- `project_command_run`

## 最佳实践
1. 先定位相关目录，再读取具体文件
2. 只读取回答问题所需的最小文件集
3. 运行命令前确认它属于允许的安全命令
4. 优先做读操作，再做写操作

## 失败处理
- 命令不在允许列表时不要强行执行
- 如果项目结构不清楚，先用代码列表工具确认

## 输出要求
- 简洁说明发现了什么
- 对命令结果给出可执行结论
